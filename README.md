
# База данных игроков спортивной команды

Данный проект реализован в рамках учебного задания (вариант №2 "Спортивная команда").  
Приложение представляет собой систему для просмотра и поиска спортсменов по базе данных, используя структуру Go-проекта и PostgreSQL.

## Стэк

* Go 
* PostgreSQL - pgAdmin
* HTML/CSS
* Fetch API
* Go net/http

## Функционал

* Подключается к локальной базе данных PostgreSQL (sportteam_db)
* Читает данные из трёх связанных таблиц: player, player_name, city
* Поднимает HTTP-сервер на localhost:8080
* Реализует REST-маршрут GET /players, возвращающий список игроков в формате JSON
* Поддерживает параметр q в запросе (/players?q=Москва), выполняя поиск по части ФИО и городу (через ILIKE)
* Отдаёт статический web-интерфейс из каталога web/static (страница index.html)
* Интерфейс отображает таблицу игроков (ФИО, город, рост, вес) и поле поиска по базе данных
* Поддерживает фильтрацию списка игроков на стороне сервера по запросу пользователя и динамическое обновление таблицы через Fetch API

## Как запустить проект 

### 1. Настройка базы данных PostgreSQL
#### 1.1 Создание пользователя и базы данных
Необходимо открыть консоль psql или PowerShell:
```sql
psql -U postgres
```
#### 1.2 Создать пользователя и базу данных:
```sql
CREATE USER sport_user WITH PASSWORD 'sport_pass123';
CREATE DATABASE sportteam_db OWNER sport_user;
```
### 2. Применение SQL-схемы

В корне проекта находится файл database.sql, который создаёт таблицы:
* player_name
* city
* player
и заполняет их демонстрационными данными.

Чтобы применить его:
```sql
psql -U postgres -h localhost -d sportteam_db -f 
```
Можно также выполнить файл через pgAdmin (Запросник → Выполнить файл).

### 3. Проверка базы данных

Подключиться под пользователем sport_user:
```sql
psql -U sport_user -d sportteam_db
```
Проверить таблицы:
```sql
\dt
```
Проверить количество игроков:
```sql
SELECT COUNT(*) FROM player;
```
### 4. Запуск Go-сервера

Перейти в каталог проекта и установить зависимости:
```sql
go mod tidy
```
Запустить сервер:
```sql
go run ./server
```
В консоли должно появится:
```sql
Подключение к БД успешно
Сервер запущен на http://localhost:8080
```
### 5. Проверить результаты

Перейди в браузере по адресу:
[http://localhost:8080](http://localhost:8080)
